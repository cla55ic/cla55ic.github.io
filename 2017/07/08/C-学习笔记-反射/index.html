程序集

.net 中 dll 和 exe 文件都是程序集，可以看作是多个相关类的集合。

程序集包含资源文件、类型元数据，每个城预计都有自己的名称、版本等信息。这些信息可以通过 AssemblyInfo.cs 文件来自定义。

反射

反射就是动态的获取程序集中的元数据。反编译工具、VS智能提示就是通过反射来实现。

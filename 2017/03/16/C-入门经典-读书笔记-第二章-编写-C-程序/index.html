[C#入门经典]读书笔记-第二章-编写 C# 程序

发表于 2017-03-16

1 Visual Studio 2012 开发环境

  • VS 2012 最常用的窗口
  1. 解决方案资源管理器 (Solution Explorer) ：显示解决方案包含的项目及文件
  2. 属性 (Properties) ：显示选中文件或元素的详细信息，并可进行更改
  3. 错误列表 (Error List) ：显示程序中的错误、警告等信息

    使用 VSS 协作开发时，挂起的签入 窗口可显示当前签出的文件及修改信息。

2 控制台应用程序

  • 创建并运行控制台应用程序
  1. 选择 文件 | 新建 | 项目 | 控制台应用程序

  2. 键入示例代码

  3. 选择 调试 | 启动调试 或按 F5 即可看到如图所示的结果

3 桌面应用程序(WPF)

  • 创建并运行桌面应用程序
  1. 选择 文件 | 新建 | 项目 | WPF 应用程序
  2. 打开 MainWindow.xaml 文件，可以分别看到 设计 和 代码 两个窗口
  3. 点击 工具箱 ，双击 Button 控件，可以看到 设计 窗口中添加了一个按钮

  4. 双击新加入的按钮，跳转到 ‘MainWindow.xaml.cs’ 文件中，键入示例代码

  5. 选择 调试 | 启动调试 或按 F5 即可看到如图所示的结果

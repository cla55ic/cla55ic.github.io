桌面应用程序(WPF)

  • 创建并运行桌面应用程序
  1. 选择 文件 | 新建 | 项目 | WPF 应用程序
  2. 打开 MainWindow.xaml 文件，可以分别看到 设计 和 代码 两个窗口
  3. 点击 工具箱 ，双击 Button 控件，可以看到 设计 窗口中添加了一个按钮

  4. 双击新加入的按钮，跳转到 MainWindow.xaml.cs 文件中，键入示例代码

  5. 选择 调试 | 启动调试 或按 F5 即可看到如图所示的结果
